SetSizeAndShape(Double, Double, Double, Boolean)

Specify the object's size and shape. This can be used for any object. The other methods are for convenience. SetSphere(Double) SetSpheroid(Double, Double) SetEllipsoid(Double, Double, Double) SetIrregularShape(Double, Double, Double)

Parameters
Type Name Description

System.Double radiusB

The second radius (or half width).
System.Double radiusC

The third radius (or half height).
System.Boolean isRound

This flag should be:

  • true for round objects (stars, planets, dwarf planets, satellite planets)
  • false for lumpy objects (small bodies, satellite planetoids)
